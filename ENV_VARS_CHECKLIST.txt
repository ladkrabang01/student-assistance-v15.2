═══════════════════════════════════════════════════════════════════════════════
SETUP STEPS IN VERCEL
═══════════════════════════════════════════════════════════════════════════════

1. ไปที่ https://vercel.com/dashboard
2. คลิก project: "student-assistance-v15-24"
3. คลิก "Settings" tab
4. คลิก "Environment Variables" ใน left sidebar
5. สำหรับแต่ละตัวแปร:
   - คลิก "Add New"
   - ใส่ Name (เช่น VITE_APP_ID)
   - ใส่ Value (ค่าจริง)
   - เลือก Environments: ✓ Production, ✓ Preview, ✓ Development
   - คลิก "Save"
6. หลังจากเพิ่มทั้งหมด:
   - ไปที่ "Deployments" tab
   - คลิก latest deployment
   - คลิก "Redeploy" button
   - รอให้ build เสร็จ (ประมาณ 2-3 นาที)

═══════════════════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

❌ Error: "Unexpected token 'A', ... is not valid JSON"
→ สาเหตุ: Missing environment variables
→ แก้: เพิ่มตัวแปรทั้งหมด แล้ว Redeploy

❌ Error: "Missing required environment variables"
→ สาเหตุ: ตัวแปรบางตัวไม่ได้ set ใน Vercel
→ แก้: ตรวจสอบรายชื่อด้านบน เพิ่มตัวที่ขาด

❌ Error: "Cannot connect to database"
→ สาเหตุ: DATABASE_URL หรือ TURSO_AUTH_TOKEN ผิด
→ แก้: ตรวจสอบค่าใน Turso Dashboard

❌ Error: "OAuth login doesn't work"
→ สาเหตุ: VITE_APP_ID หรือ OAUTH_SERVER_URL ผิด
→ แก้: ตรวจสอบค่าจาก Manus

❌ Error: "Cannot authenticate"
→ สาเหตุ: JWT_SECRET ขาด
→ แก้: Generate JWT_SECRET ใหม่ด้วย: openssl rand -base64 32
